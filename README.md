# Coding Convention
Similar to C++ STL with exceptional rule for compound properties/variables.
  * my_namespace
  * my_class
  * my_method
  * my_primitive (maths is using both lower-case and upper-case)
  * my_Compound (maths is using both lower-case and upper-case)
  * MY_CONSTANT
